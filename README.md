Презентация на тему "Физика" в учебных целях
===========================

Требования
----------

- Презентация должна быть выполнена с помощью Markdown, где это возможно
- Презентация должна быть выполнена с помощью фреймворка Reveal.JS
- Тема (оформление) обязательно должна быть вынесена в отдельный CSS файл
- Минимальное количество слайдов: 20
- Перед каждым разделом необходимо использовать слайд-разделитель
- Оформление презентации должно быть читаемым
- Обязательными являются титульный слайд и заключительный слайд
- Обязательно наличие меню для быстрого перехода к слайду с помощью плагина reveal.js-menu
- Обязательно наличие навигации: стрелки и номер слайда
- Обязательно наличие readme в корне презентации с заданием без указания баллов
- На слайде не должно быть скроллов
- Минимальное количество изображений в презентации: 3
- Необходимо использовать локальную версию фреймворка Reveal.js

Обязательные слайды
-------------------

- Единицы измерения для категорий: скорость, температура, длина, масса, площадь, объем, давление
- Для каждой категории должно быть не менее 5 единиц измерения с обозначением и описанием применения
- Программы, написанные на языке JavaScript, для перевода величин из одной системы измерения в другую
- Необходимо сделать конвертеры для 4 категорий из перечисленных в презентации
- Слайды с задачами на скорость, время и расстояние
- Описание и примеры решения задач на скорость, время и расстояние
- Программа на JavaScript для решения задачи на скорость, время и расстояние
- На вход программы должны подаваться 3 поля: скорость, время, расстояние
- Для расчета третьего значения должны быть заполнены любые 2 поля
- Исходный код программ
- При наличии оставшихся слайдов, но не менее 3, их можно заполнить картинками по теме
